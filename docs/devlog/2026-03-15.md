# 2026 March 03 - First Entry
Decided to start using GitHub Pages and to create the first entry of this devlog.  
First odd thing I noticed is that GitHub Pages by default uses the light theme, which I don't like. So let's figure out what I have to do to change it to "dark theme".
Mistral's Le Chat and [this Stack Overflow post](https://stackoverflow.com/questions/70538816/use-the-dark-mode-equivalent-of-the-github-pages-default-theme) make me believe that just creating a YML file with this single line in it should be enough to change the theme for this repo:
```yml
theme: jekyll-theme-dark
```

Let's see if it works.
